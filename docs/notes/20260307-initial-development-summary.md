# 作業要約: サッカー作戦ボードアプリの初期開発

**作業日時**: 2026年3月7日
**担当**: Antigravity

## 議論・作業の概要
ユーザーの要望に基づき、サッカーの作戦ボードアプリ「Soccer Tactical Board」を新規開発しました。
Tacticalistaという既存サービスを参考にし、React + Vite + Konva (Canvas) の技術スタックを採用しました。

## 決定事項・合意内容
- **技術選定**: 型安全な開発のために TypeScript を使用。複雑な描画を実現するため `react-konva` を採用。
- **デザイン**: ダークテーマのツールバーと、白地に黒線のピッチデザイン（ユーザー指定のビジュアル）。
- **機能範囲**: 選手のドラッグ、自由線・直線・矢印の描画、テキスト入力（カスタムモーダル）、フォーメーション（4-4-2）の自動配置、画像出力、ブラウザへの自動保存。

## 重要な設計メモ
- `App.tsx` 内で、描画用の `lines` とオブジェクト用の `objects` を独立した配列で管理しています。
- ペナルティ・アークの描画は `Arc` コンポーネントを使用し、ペナルティエリアの境界線に接するように角度を調整（角度 106度、回転 37度/217度）。
- Konva の `Stage` から `toDataURL` を使用して PNG 書き出しを実現しています。
